BMI Calculator

A simple Python project that calculates the Body Mass Index (BMI) based on user input weight and height.  The program then categorizes the BMI result and provides relevant health suggestions.

Features
•	Accepts user weight and height input
•	Calculates BMI accurately using formula: BMI = weight / (height * height)
•	Classifies health status:
1.	Underweight
2.	Normal
3.	Overweight
4.	Obese
5.	Clinically Obese
6.	Provides helpful health suggestions
7.	Handles invalid input
8.	Allows multiple calculations in one run


Technologies Used
•	Python 

How to Run
1. Make sure Python is installed on your system.
2. Download the project file (`bmi_calculator.py`).
3. Open a terminal or command prompt in the project directory.
4. Run the script using: python bmi_calculator.py
